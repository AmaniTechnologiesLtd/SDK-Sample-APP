Redirect User to Web SDK

After you created the token for single user. You can redirect user to amani Web SDK page with created token. You should add params below like below.

https://client.amani.ai/?tk={token}&id={CustomerIdentiyCardNumber}&rl={returnLink}&bu={baseUrl}&ln={languageCode}
